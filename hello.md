## Git CLI 명령어들

clone: 원격 저장소를 로컬 컴퓨터에 복사
add: 스테이지 영역에 직접 파일 추가
commit: 스테이지 영역위 파일들을 가지고 커밋
Push: 커밋한 내용을 원격 저장소에 업로드
